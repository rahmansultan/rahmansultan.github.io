KwhKosKita

Deskripsi :

KwhKosKita adalah platform monitoring yang dikembangkan menggunakan CodeIgniter 3, PHP, dan MySQL. Platform ini didesain untuk menampilkan data dari alat monitoring kWh listrik berbasis IoT. Proyek ini mengintegrasikan sensor PZEM dan NodeMCU ESP8266 untuk mengumpulkan data secara real-time. Pengguna dapat memantau dan menganalisis konsumsi listrik di suatu lokasi, memberikan wawasan berharga untuk efisiensi energi dan manajemen daya.

Peran :
  • Perancang Alat / IoT
  • Pengembang Platform / Web

KharismaTailor

Deskripsi :

KharismaTailor merupakan sistem manajemen data pelanggan yang dibangun dengan menggunakan CodeIgniter 3 dan MySQL. Sistem ini dikembangkan khusus untuk memenuhi kebutuhan bisnis tailor bernama Kharisma. Dengan KharismaTailor, pemilik bisnis dapat mengelola informasi pelanggan, seperti nama, alamat, nomor telepon, dan tanggal pendaftaran. Sistem ini juga menyediakan fitur edit dan hapus pelanggan untuk memudahkan pembaruan dan pemeliharaan data.

Peran :
  • Pengembang Utama Platform / Web

Pesan Wisata Alam

Deskripsi :

Proyek Pesan Wisata Alam adalah platform pembelian tiket wisata yang dirancang menggunakan CodeIgniter 3 dan MySQL. Platform ini menyediakan layanan pembelian tiket untuk destinasi wisata alam tertentu. Pengguna dapat menjelajahi berbagai pilihan destinasi, memilih tanggal perjalanan, dan memesan tiket secara online. Sistem ini juga dapat menyimpan riwayat pemesanan pengguna, memudahkan mereka untuk melacak dan mengelola perjalanan wisata mereka.

Peran :
  • Pengembang Front-End Platform / Web
